# 3. Zasada: minimum stabilności > maksimum ambicji
Po epizodzie maniakalnym najgroźniejsze bywa:


- zarywanie nocy,
- skakanie między projektami,
- nadmierna stymulacja,
- kofeina + internet + “mam OLŚNIENIE”.


**Mózg uwielbia wtedy robić:**

“A może zostanę twórcą aplikacji, muzykiem ambientowym i nomadem cyfrowym jednocześnie?”
Nie. Siadasz. Pijesz wodę. Robisz jeden commit.

# 4. Stwórz “system tygodnia”
Nie plan życia. Plan tygodnia.
Codziennie tylko:

- 1 rzecz dla zdrowia,
- 1 rzecz dla nauki,
- 1 rzecz dla utrzymania życia.

**Przykład:**

- spacer,
- 30 min Git/GitHub,
- pranie.

To już jest dzień wygrany. Naprawdę. Internet ci wmówił, że trzeba codziennie “grindować”. Internet też uważa, że ludzie chcą oglądać 6-godzinne analizy lore ekspresów do kawy, więc zachowaj dystans.

# 5. Nauka GitHuba — prosty roadmap
Nie ucz się wszystkiego naraz.
Kolejność:

- repozytoria,
- commit,
- push/pull,
- branches,
- merge,
- README,
- .gitignore,
- pull requesty.


I rób mikroskopijne projekty:

- kalkulator,
- lista zadań,
- tracker nastroju,
- markdown journal.

**Małe projekty budują poczucie kierunku lepiej niż binge oglądania tutoriali o “architekturze mikroserwisów”. Człowiek obejrzy 14 godzin materiału i potem stworzy folder projekt_final_v2_REAL.**

# 6. Najważniejsze: nie utożsamiaj chaosu z tożsamością
To, że miałeś epizod maniakalny, nie oznacza:

- że jesteś “zepsuty”,
- ani że całe życie będzie niestabilne.

Ale oznacza, że warto traktować swój stan poważnie i budować strukturę wcześniej, a nie dopiero podczas pożaru.
Ludzie często chcą “wrócić do dawnej wersji siebie”. Może lepiej stworzyć spokojniejszą wersję 2.0. Mniej spektakularną. Za to działającą bez crashowania co trzy tygodnie jak źle napisany frontend.
I serio — jeśli chcesz, mogę ci też pomóc zrobić:

- prosty tygodniowy system życia,
- roadmap nauki programowania,
- minimalistyczny Notion/GitHub workflow,
- albo nawet strukturę “repozytorium życia”. Bo najwyraźniej zostałem cyfrowym opiekunem ludzi próbujących odzyskać sterowność. Fascynujące.
